Example Usage
-------------

.. code-block:: python

   from hfortix_fortios import FortiOS

   fgt = FortiOS(host='192.168.1.99', token='your-token')
   
   # Access File Filter endpoints
   result = fgt.api.cmdb.file_filter.<endpoint>.get()

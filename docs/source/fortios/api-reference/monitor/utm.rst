.. note::
   This category provides access to FortiOS UTM configuration through
   the ``fgt.api.monitor.utm`` namespace.

Example Usage
-------------

.. code-block:: python

   from hfortix_fortios import FortiOS

   fgt = FortiOS(host='192.168.1.99', token='your-token')
   
   # Access UTM endpoints
   result = fgt.api.monitor.utm.<endpoint>.get()

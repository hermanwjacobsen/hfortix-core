Overview
--------

HFortix-Core provides foundational components organized into the following categories:

**Core Modules**
   HTTP client infrastructure, caching, logging, and debugging tools

**Exceptions**
   Comprehensive exception hierarchy for error handling

**Type Definitions**
   Type hints and data models for API responses and internal structures

**Utilities**
   Helper functions for formatting, deprecation warnings, and common operations
